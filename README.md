# 🍄 Guía de Hongos Adaptógenos — FungiPharma

Guía interactiva de los 7 extractos de hongos funcionales de [fungipharma.cl](https://fungipharma.cl).

## Demo

Abre `index.html` en tu navegador — no requiere servidor ni dependencias.

## Características

- Carrusel interactivo con los 7 hongos
- Navegación con flechas, dots y teclado (← →)
- Sheet de propiedades con info real de cada producto
- Diseño responsivo desktop + mobile
- Standalone — todo embebido en un único HTML (imágenes en base64)

## Uso

```bash
# Clonar
git clone https://github.com/TU_USUARIO/fungipharma-web.git

# Abrir en Chrome
open index.html
```

## Tecnología

- HTML + CSS + JavaScript vanilla
- Fuentes: Cormorant Garamond + DM Mono (Google Fonts)
- Sin frameworks, sin dependencias

## Despliegue en GitHub Pages

1. Ve a **Settings → Pages**
2. Source: `Deploy from a branch`
3. Branch: `main` / `root`
4. Guarda — tu guía estará en `https://TU_USUARIO.github.io/fungipharma-web`
